This function takes one number input and outputs its list of prime factors

example: prime_factors(22)
        [2, 11]
        
example: prime_factors(22343)
        [22343]
        
example: prime_factors(359723589)
        [2, 241, 647, 769]
        
example: prime_factors(56)
        [2, 2, 2, 7]
